# Telemt Docker Installer

Интерактивный скрипт для автоматического развертывания **Telemt** в Docker на популярных Linux дистрибутивах.

## ✨ Возможности

- 🚀 **Полная автоматизация** — установка Docker, генерация конфига, запуск контейнера
- 🖥️ **Поддержка дистрибутивов** — Ubuntu, Debian, Fedora, RHEL, CentOS, Rocky Linux, AlmaLinux, openSUSE, Arch Linux
- 🔐 **Автогенерация секрета** — случайный секрет доступа (или свой)
- 🔗 **Автоопределение URL** — скрипт сам покажет готовую ссылку для подключения

## 📋 Требования

- Linux сервер (см. список поддерживаемых ОС)
- Права root (sudo)
- Домен, направленный на ваш сервер (для TLS)
- Доступ в интернет

## 🚀 Быстрый старт

```bash
# 1. Скачать скрипт
curl -O https://raw.githubusercontent.com/qreboot/amneziawg-go-docker/main/setup-telemt.sh

# 2. Сделать исполняемым
chmod +x setup-telemt.sh

# 3. Запустить полную установку
sudo ./setup-telemt.sh --install
```

## 📖 Что вас спросят во время установки
| Вопрос | Что отвечать |
|--------|--------------|
| Домен для TLS |	Ваш домен, например vpn.example.com (обязательно) |
| Секрет доступа |	Оставьте пустым — сгенерируется автоматически |
| Ad tag |	Метка для рекламы, опционально — можно оставить пустым |
| Порт сервера |	Нажмите Enter для порта 443 (стандартный HTTPS) |
| Публичный адрес |	IP или домен сервера (обычно совпадает с доменом TLS) |

## 🎯 Что вы получите

   - ✅ Работающий Telemt сервер в Docker

   - ✅ Конфиг в /srv/containers/telemt/config/config.toml

   - ✅ Готовую ссылку для подключения вида:
    https://t.me/proxy?server=ПУБЛИЧНЫЙ_АДРЕС&port=ПОРТ_СЕРВЕРА&secret=somelongsecretstring

## 📖 Использование
### Интерактивное меню
```bash

sudo ./setup-telemt.sh
```

Меню позволяет:

   - Полная установка

   - Только установка Docker

   - Только генерация конфигурации

   - Запустить контейнер

   - Остановить контейнер

   - Перезапустить контейнер

   - Показать информацию о подключении

   - Обновить секрет доступа

### Командная строка
```bash

# Полная установка
sudo ./setup-telemt.sh --install

# Показать ссылку для подключения
sudo ./setup-telemt.sh --show-info

# Сгенерировать новый секрет
sudo ./setup-telemt.sh --regenerate-secret
```
### 🐳 Управление контейнером
```bash

# Просмотр логов
docker logs -f telemt

# Остановка
docker stop telemt

# Запуск
docker start telemt

# Перезапуск
docker restart telemt

# Статус
docker ps | grep telemt
```
### 📁 Структура файлов
```text

/srv/containers/telemt/
└── config/
    └── config.toml      # Конфигурационный файл
```
### 🔧 Устранение неполадок
#### Контейнер не запускается
```bash

# Проверить логи
docker logs telemt

```
#### Ошибка "port already allocated"

Порт 443 может быть занят другим сервисом. Укажите другой порт при установке.
#### Нет ссылки для подключения

Дайте контейнеру 10–20 секунд на запуск, затем выполните:
```bash

sudo ./setup-telemt.sh --show-info
```
